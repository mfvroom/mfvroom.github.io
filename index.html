How to Apply CSS to HTML

The most common way of applying CSS to a document is by referencing an external CSS stylesheet from an HTML element. The code for the CSS sytlesheet would something like this:
h1 {
color: blue;
}
p {
color: green;
}

How to Use JavaScript in a Webpage

Adding JavaScript in a webpage utlizes the HTML tag script, which wraps around the JavaScrpit code. The code for the JavaScript would look something like this:

let d = new Date();
alert("The current date and time is: " + d);

Remember this code would need to be enclosed in the script tags.
